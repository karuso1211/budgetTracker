# Changelog

## v1.x

### Debt Tracker
- New **Debt** tab to log what you owe (name + amount)
- Tap a debt card to expand — pay in full or partially
- Payment history tracked inside each card with an **Undo** button per entry
- Debt auto-marked complete when fully paid; creation date recorded

### Budget Tracker
- Removed **Shopping** and **Bills** from expense categories

### Savings
- Adding a saving with an existing name now adds to it instead of creating a duplicate entry
- Savings card always visible in Overview even at ₱0.00

### Spending Stats
- New **Spending Stats** tab with an animated bar chart
- Toggle between **Daily** (7 days), **Weekly** (8 weeks), and **Monthly** (6 months) views
- Summary row shows total, average, and peak for the selected period
- Quick-view card on Overview shows total spent + top 3 categories — tap it to jump to the tab

### History
- Expense entries now show their category icon next to the name
